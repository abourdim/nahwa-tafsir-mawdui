# 🌊 Toward Thematic Quran Interpretation — نحو تفسير موضوعي لسور القرآن

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **نحو تفسير موضوعي لسور القرآن** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 cards on thematic Quran interpretation, 8 reflections on interpretation challenges, daily habit tracker with streaks, self-assessment quiz
- Trilingual: Arabic, English, French with RTL/LTR support
- 3 unique themes with localStorage persistence
- Sound effects, responsive design, PWA-ready

## Tech

Pure HTML + CSS + JS. No frameworks, no dependencies. Just open `index.html`.

## Disclaimer

I am not a scholar. This is a humble effort by a Muslim parent to share Sheikh al-Ghazali's wisdom interactively. Content derived from the book and trusted Islamic sources. This is not a fatwa.
